# ⌨️ Using Windows Command Line

#### 1. What command corresponds to the alias 'HoldenManeuver'?
> **Answer:**  `Get-Runspace`  
```powershell
alias HoldenManeuver
```

#### 2. How many books are found under Documents\Books?
> **Answer:**  `9`
```powershell
cd .\Documents\Books\; Get-ChildItem
```
or
```powershell
Get-ChildItem .\Documents\Books\
```

#### 3. Which cmdlet is used to display a list of processes on the system?
> **Answer:**  `Get-Process`

#### 4. How many services have 'MCRN' in their name?
> **Answer:**  `5`
```powershell
Get-Service | Where-Object -Name "*MCRN*"
```

#### 5. How many active users are there in the Active Directory environment?
> **Answer:**  `10`
```powershell
Get-ADUser -Filter * | Where-Object { $_.Enabled -eq $true } | Measure-Object
```
#### 6. Which local group has a description mentioning certificates?
> **Answer:**  `Cert Publishers`
```powershell
Get-LocalGroup | Where-Object { $_.Description -like "*certificates*" }
```
#### 7. Which command can be used to download files from the internet or another computer on the network using PowerShell?
> **Answer:**  `Invoke-WebRequest`

#### 8. What is the build number of the system?
> **Answer:**  `17763`
```powershell
systeminfo
```
#### 9. What is the HotFixID of the installed update on the system?
> **Answer:**  `KB4464455`

```powershell
Get-HotFix
```

#### 10. Is Defender running on the server?

#### 11. Which user has read-only permission to the file named Abaddon's Gate under Books?
> **Answer:**  `c.avasarala`
```powershell
Get-Acl "Abaddon's Gate" | Select-Object -ExpandProperty Access
```
